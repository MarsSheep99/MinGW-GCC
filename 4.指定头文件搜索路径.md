## 4.指定头文件搜索路径

#### 一、头文件绝对路径

```
gcc *.c -I <头文件绝对路径（即具体路径）> 
例如：gcc *.c -I C:\Visual_Studio_Code_Project\head
```

#### 二、头文件相对路径

```
gcc *.c -I ../<头文件所在文件夹>
例如：gcc *.c -I ../head
..表示当前文件夹的上一级文件夹，.表示当前文件夹。
```

#### 三、任意路径编译

```
gcc <源文件所在文件夹>/*.c -I <头文件所在文件夹>
例如：gcc ../practice/*.c -I ../head
```
